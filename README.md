# Cat's Rage

## Setup

1. `git clone https://github.com/AlfreMK/cats-rage.git`
2. In Unity Hub, open the folder `cats-rage` (it will take a while to download the packages)
3. In Unity Editor, open the scene `Assets/Scenes/MainScene.unity`

## Controls

### First player
- `WASD`: Move
- `V`: Shoot
- `G`: Punch

### Second player
- Arrow keys: Move
- `M`: Shoot
- `J`: Punch
